## Primer Programa

[Primer Programa](../src/hola_mundo.py)

```python
print("Hola Mundo")
print(1)            
```

_Modo de ejecucion de script_

```shell
python3 hola_mundo.py
```

_Salida_

```shell
Hola Mundo
1
```
