## Operadores de Incremento o Decremento

```python
# Incremento
year = 2020

year = year + 1
# year += 1
print(year)

# Decremento
year = year - 1
# yeat -= 1
print(year)
```

## Operadores de Comparacion

* "=="
    * Igual que
* "!="
    * Diferente de
* "<"
    * Menor que
* ">"
    * Mayor que
* "<="
    * Menor que o Igual que
* ">="
    * Mayor que o Igual que

_Comprobacion con una sentencia de control_

```python
color = "rojo"

if color == "rojo":
    print("El color es rojo")
else:
    print("El color no es rojo")
```
