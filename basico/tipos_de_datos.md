
# Tipos de Datos

[Tipos de Datos](../src/td_datos.py)

* None
    * Es un tipo de dato vacio, nada
* Cadena de Texto
    * Una cadena de texto
* Entero
    * Un numero entero
* Flotante
    * Un numero decimal
* Booleano
    * Verdadero o Falso
* Lista
    * Un conjunto de Datos
* Tupla 
    * Un lista pero inmutable
* Diccionario
    * Un conjunto de datos con clave y valor
* Rango
    * Establece un rango, una secuencia numerica
